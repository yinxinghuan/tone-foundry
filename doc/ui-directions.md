> 本文只记录候选方向，不替代当前 `doc/visual.md`。用户选定方向后，再将胜出方案写入正式视觉文档并重做生产界面。

## 评审结论

- 2026-07-16：选择 **C — 声音标本馆 / Sonic Index** 进入生产化。
- 正式产品以英文为主语言、中文为完整辅助语言。评审入口默认显示英文 C 方案，并提供 `EN / 中文` 即时切换，用于提前检查英文标题、模块名和操作按钮的移动端长度。

## 共同界面约束

- 主平台：手机竖屏 `360–430 px`，触摸操作，所有触控目标至少 `44×44 px`。
- 每套方向必须展示入口、制琴选择、音色测试和成琴档案，而不是只做封面。
- 吉他始终是第一视觉焦点；选件与试听处于同一画面。
- 未完成的吉他仍以结构轮廓表达，选择后即时显影。
- 不沿用当前生产界面的收据、蓝墨印刷、检验章、楷体手写和蓝色金属按钮。
- 不使用 Emoji、玻璃拟态、霓虹渐变或通用 SaaS 卡片。
- 评审页只改变视觉与信息结构，不改抽取概率、模块兼容、音色逻辑和 Riff 机制。

## A — 夜间制琴室 / Nocturne Atelier

- **概念**：现代精品制琴品牌的私人试听室，安静、深色、收藏级。
- **构图**：吉他占屏幕约 62%，底部是一条悬浮但有实体厚度的选件托盘；信息尽量少。
- **形状**：长直线、窄边、克制圆角、金属拨轮；不使用纸张。
- **材质与光**：煤黑漆面、烟熏玻璃只作为非文字背景、深胡桃木、拉丝镍和一束暖工作灯。
- **颜色**：Carbon `#101111`、Warm Black `#191A18`、Ivory `#F0E9DC`、Nickel `#A8AAA6`、Amber `#C9954E`。
- **排版**：现代无衬线为主，英文使用 Helvetica Neue / Arial，中文使用 PingFang SC；大标题轻字重，不用装饰字体。
- **标志性瞬间**：选中部件时，工作灯只扫过对应部位，底部参数刻度缓慢落位。
- **风险**：过度克制会接近高端商品页，因此必须保留可玩性和部件显影。

## B — 模块唱片社 / Modular Groove

- **概念**：七十年代唱片内页、音乐器材广告和模块玩具的结合，直接、鲜活、有收藏冲动。
- **构图**：大胆分区，吉他可以偏轴放置；候选是大面积色块与编号，不追求对称。
- **形状**：圆角方块、胶囊只用于标签、粗线和实心图形；结构像唱片封套，不像后台。
- **材质与光**：哑光印刷色、半透明塑料、黑色橡胶和镀铬旋钮；拟物集中在可操作件。
- **颜色**：Cream `#F3E7CE`、Ink `#151515`、Tomato `#E54A32`、Cobalt `#315BD8`、Lemon `#EFCB43`。
- **排版**：粗体 grotesk 与紧凑数字，中文使用 PingFang SC Heavy；不使用衬线或手写。
- **标志性瞬间**：选择部件时，编号色块像唱片机按键一样压下，琴体同步切换并短暂放大。
- **风险**：颜色过多会抢吉他，因此同时只允许一个主色和一个辅助色活跃。

## C — 声音标本馆 / Sonic Index

- **概念**：当代艺术书、工业设计目录和乐器博物馆标签组成的清醒编辑系统。
- **构图**：大量留白、明确基线、吉他像标本居中陈列；音色数据与模块名称沿边缘排列。
- **形状**：几乎无圆角，细线、索引号、裁切标记和窄列；不模拟旧纸或档案袋。
- **材质与光**：暖白无涂层纸、黑色油墨、冷灰铝板和柔和自然光，实体深度很浅。
- **颜色**：Paper `#F2F0EA`、Graphite `#181917`、Steel `#A7AAA6`、Signal Green `#B8D63D`、Rust `#A74C32`。
- **排版**：瑞士式无衬线与等宽数字；中文使用 PingFang SC，标题使用紧凑字距，不使用宋体或楷体。
- **标志性瞬间**：部件变化时，索引线从模块名称延伸到吉他对应位置，音色条同步重新排序。
- **风险**：过度理性会显得冷，需要通过真实吉他材质和声音反馈保持情感。

## 评审问题

1. 哪一套最像你希望长期收藏和创作的游戏，而不只是一次性漂亮页面？
2. 哪一套能让吉他成为第一焦点，同时仍愿意主动打开音色测试？
3. 哪一套更适合未来加入木吉他、超现实模块和公共琴墙？
4. 是否需要把某一方向的排版与另一方向的材质组合，还是应保持单一方向完整性？
